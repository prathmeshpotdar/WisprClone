# Wispr Flow Clone

A cross-platform voice-to-text desktop app built with **Tauri** and **Deepgram**.

## Features
- Push-to-talk voice input
- Global shortcut (Ctrl + Shift + Space)
- Real-time transcription
- Clean notes UI
- Cross-platform (Windows/macOS/Linux)

## Tech Stack
- Tauri (Rust)
- React + Vite
- Deepgram Streaming API

## Run locally
```bash
npm install
npm run tauri dev
